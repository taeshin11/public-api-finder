How to Use / FAQ

Step-by-Step Guide

  1. 1
    Browse by Category

    Use the sidebar (desktop) or horizontal tabs (mobile) to filter APIs by category like Weather, Finance, Games, and more.

  2. 2
    Search for APIs

    Type in the search bar to instantly filter APIs by name, description, or category. Results update as you type.

  3. 3
    View API Details

    Each card shows the API name, description, authentication type, HTTPS support, and CORS compatibility.

  4. 4
    Save Favorites

    Click the heart icon on any card to save it to your favorites. View all favorites by clicking the Favorites tab.

  5. 5
    Discover Random APIs

    Click "Discover Random API" to find unexpected APIs for inspiration. Use "Roll Again" for more.

Frequently Asked Questions

Is Public API Finder free to use?

Yes, absolutely! Public API Finder is completely free. No account or registration is required.

Where does the API data come from?

The data is derived from the popular public-apis GitHub repository and is served as a static JSON file for fast performance.

Are my favorites saved?

Yes, favorites are saved in your browser's localStorage. They persist across sessions but are not synced across devices. You can also export your favorites as a JSON file.

What do the badges on API cards mean?

Auth: Shows the authentication type - None (no auth needed), API Key (requires a free key), or OAuth (requires OAuth flow).

HTTPS: Whether the API supports secure HTTPS connections.

CORS: Whether the API supports Cross-Origin Resource Sharing, important for frontend JavaScript calls.

Can I suggest a new API?

We welcome API suggestions! Send an email to taeshinkim11@gmail.com with the API name and link.

What languages are supported?

The UI supports 8 languages: English, Korean, Japanese, Chinese (Simplified), Spanish, German, French, and Portuguese. The language is auto-detected from your browser settings, or you can change it using the dropdown in the header.
