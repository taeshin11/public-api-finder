Privacy Policy

Last updated: April 1, 2026

1. Information We Collect

Public API Finder is committed to protecting your privacy. We collect minimal data:

  • Anonymous visitor counts - We track page visits without storing IP addresses or personal identifiers.
  • Search queries - Anonymized search terms may be logged to improve the service. No personal information is attached.
  • Click-through data - We may record which APIs are clicked (API name only) for analytics.
  • Language preference - Stored locally in your browser via localStorage.

2. Local Storage

We use browser localStorage for the following purposes:

  • Favorites - Your saved API favorites are stored locally on your device and never transmitted to any server.
  • Language preference - Your selected interface language.
  • Visitor statistics - Anonymous visit counter data.

You can clear this data at any time through your browser settings.

3. Cookies

Public API Finder does not set any first-party cookies. Third-party advertising partners (Adsterra, Google AdSense) may set cookies for ad personalization. You can manage cookie preferences in your browser settings.

4. Third-Party Services

We use the following third-party services:

  • Google AdSense - For displaying advertisements. Google's privacy policy applies.
  • Adsterra - For displaying advertisements. Adsterra's privacy policy applies.
  • Vercel - For website hosting. Vercel's privacy policy applies.
  • Google Sheets - For anonymous analytics data collection via webhook.

5. Data Security

We take reasonable measures to protect the limited data we collect. All communications with our website are encrypted via HTTPS. We do not store any personally identifiable information (PII).

6. Children's Privacy

Public API Finder does not knowingly collect personal information from children under 13. The service is intended for developers and students of all ages.
