About Public API Finder

What is Public API Finder?

Public API Finder is a free, searchable, and categorized directory of public APIs designed for developers, hobbyists, and students. Our goal is to make it easy to discover the perfect API for your next project.

Whether you are building a weather app, a financial dashboard, a trivia game, or just looking for inspiration, we have organized 200+ APIs across 15+ categories to help you find exactly what you need.

Features

  • Category Browsing - Navigate APIs by category using the sidebar or horizontal tabs.
  • Instant Search - Filter APIs in real-time as you type.
  • Favorites - Save your favorite APIs with localStorage persistence.
  • Random Discovery - Find unexpected APIs with the Random API button.
  • 8 Languages - Available in English, Korean, Japanese, Chinese, Spanish, German, French, and Portuguese.

Data Source

Our API directory is derived from the popular public-apis GitHub repository, curated and organized for easy discovery. The data is served as a static JSON file, ensuring zero API costs and lightning-fast performance.
